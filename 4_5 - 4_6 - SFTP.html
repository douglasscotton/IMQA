4.5 Controle de Qualidade do Processo

Além dos conceitos de CQ automatizados pelo software de captura, a Iron Mountain utiliza uma  ferramenta de CQ, suportada pela norma NBR 5426, que aplica planos de amostragem e procedimentos  na inspeção de atributos. Realizaremos o controle de qualidade em 5% dos documentos para validar os  seguintes atributos: 
 

  • Ausência de imagem: serão identificados os documentos físicos pendentes de digitalização e será  feito a inserção das imagens no processo. 
  • Imagem ilegível ou com sobreposição de informação: são identificados os documentos físicos e  separados para nova captura. As imagens anteriores serão excluídas. 
  • Indexação dos documentos: serão validados todos os indexadores cadastrados.
 




 
 













 
 
 
 

4.6 Disponibilização das imagens via Link SFTP

É o portal de acesso às imagens, ferramenta onde a CONTRATANTE irá acessar as imagens digitalizadas. O download das imagens poderá ser realizado em até 30 dias. 

O SSH File Transfer Protocol (também conhecido como Secure File Transfer Protocol, ou SFTP) é um protocolo de rede que fornece acesso a arquivos, transferência e gerenciamento de arquivos em um fluxo de dados confiável. Ele foi projetado pela Internet Engineering Task Force (IETF) como uma extensão do protocolo Secure Shell (SSH) versão 2.0 para fornecer recursos de transferência segura de arquivos. Esse protocolo pressupõe que seja executado em um canal seguro, como SSH, que o servidor possa autenticar o cliente e que a identidade do usuário esteja disponível para o protocolo.
